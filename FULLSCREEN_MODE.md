# Fullscreen Mode

JustWrite now includes a true fullscreen mode for complete immersion in your writing!

## ðŸ–¥ï¸ What is Fullscreen Mode?

Fullscreen mode takes over your entire screen, hiding:
- Window title bar and borders
- Windows taskbar
- Desktop background
- All other applications
- System notifications (when focused)

**Result:** Just you and your words. Complete distraction-free writing.

## âŒ¨ï¸ How to Use

### Enter Fullscreen
Press **`F11`**

### Exit Fullscreen
Press **`F11`** again or **`Escape`**

## ðŸŽ¯ Best Practices

### For Maximum Focus
1. Start JustWrite
2. Press `F11` to enter fullscreen
3. Optional: Enable Focus Mode (`Ctrl+F`)
4. Optional: Switch to Dark Mode (`Ctrl+D`)
5. Write without distractions!

### Quick Access While in Fullscreen
All keyboard shortcuts still work:
- `Escape` - Exit fullscreen (or show file menu if not in fullscreen)
- `Ctrl+S` - Save As
- `Ctrl+O` - Open
- `Ctrl+,` - Settings
- `Ctrl+D` - Dark/Light mode
- `Ctrl+F` - Focus mode

## ðŸ’¡ Tips

### The Ultimate Distraction-Free Setup
Combine these features for maximum immersion:

1. **Fullscreen** (`F11`) - Hides everything
2. **Dark Mode** (`Ctrl+D`) - Easy on the eyes
3. **Focus Mode** (`Ctrl+F`) - Highlights current sentence
4. **Large Font** (Settings) - Comfortable reading
5. **Auto-Save** (automatic) - Never worry about losing work

### Writing Sessions
- Enter fullscreen at the start of your writing session
- Use `Escape` briefly if you need the file menu
- Stay in fullscreen until you're done
- Exit with `F11` or `Escape`

### Multi-Monitor Setup
- Fullscreen takes over one monitor
- Use other monitors for research, notes, etc.
- JustWrite fills the selected screen completely

## ðŸ”§ Technical Details

### System UI
Fullscreen mode uses `SystemUiMode.immersiveSticky`:
- Hides all system UI elements
- Persists even after interactions
- Restores automatically when you exit

### Window Behavior
- App becomes borderless
- Takes full screen dimensions
- No taskbar, no window chrome
- Pure canvas for writing

## âš ï¸ Notes

### Exiting the App
- Press `Alt+F4` (Windows) to close the app
- Or press `Escape` to exit fullscreen first, then close normally

### File Operations
- All file operations work in fullscreen
- `Ctrl+S` opens save dialog
- `Ctrl+O` opens file picker
- Dialogs appear on top of fullscreen

### Settings Access
- Press `Ctrl+,` to open settings
- Settings dialog appears in fullscreen
- Close settings to return to writing
