# Font Files

Please download the following font files and place them in this directory:

## Required Fonts

### 1. Lora (Serif - Traditional Novel Feel)
Download from: https://fonts.google.com/specimen/Lora
Files needed:
- `Lora-Regular.ttf`
- `Lora-Bold.ttf`

### 2. IBM Plex Mono (Monospace - Draft/Code Feel)
Download from: https://fonts.google.com/specimen/IBM+Plex+Mono
Files needed:
- `IBMPlexMono-Regular.ttf`

## Installation Steps

1. Visit the Google Fonts links above
2. Click "Download family" for each font
3. Extract the downloaded ZIP files
4. Copy the specific .ttf files listed above to this directory
5. Run `flutter pub get` to ensure fonts are properly linked

## Font Licensing

All fonts from Google Fonts are open source and free to use.
