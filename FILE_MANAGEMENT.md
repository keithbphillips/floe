## üìã Using the File Menu

### Show the File Menu
Press **`Escape`** to show the file menu in the top-left corner.

The menu shows:
- **New** - Create a new document
- **Open** - Browse and open .md or .txt files
- **Save As** - Choose where to save your document
- **Current filename** - Shows which file you're editing
- **Unsaved indicator** - Blue dot if changes need saving

### Hide the File Menu
Press **`Escape`** again to hide the menu and return to distraction-free writing.

## üíæ How Saving Works

### Auto-Save (Background)
- Saves every 3 seconds automatically (customizable in settings)
- Creates timestamped files in `Documents/JustWrite/`
- Format: `draft_2025-11-25T10-30-45.md`
- Never lose your work!

### Manual Save As (Ctrl+S)
- Choose your own filename
- Save anywhere on your computer
- Future auto-saves use this location
- Markdown format (.md) recommended

## üìÇ Supported File Formats

### Reading
- **Markdown** (.md) - Full support
- **Plain Text** (.txt) - Full support

### Writing
- **Markdown** (.md) - Recommended
  - Compatible with GitHub, Obsidian, Notion
  - Future-proof plain text format
  - Supports basic formatting (*italic*, **bold**, # headers)

## üîÑ Workflow Examples

### Starting Fresh
1. Open JustWrite
2. Start typing (auto-saves automatically)
3. Press `Ctrl+S` when ready to choose a name
4. Pick location and filename
5. Continue writing!

### Opening Existing Work
1. Press `Ctrl+O`
2. Browse to your document
3. Click Open
4. Continue writing!

### Working on Multiple Documents
1. Press `Ctrl+S` to save current work
2. Press `Ctrl+N` for new document
3. Or press `Ctrl+O` to open another file
4. Switch between documents as needed
